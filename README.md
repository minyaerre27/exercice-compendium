# Project

This project is the first practice in the "Reproducibility in science" course frome CESAB. The goal is to create a simple research compendium.

## Content

This project is structured as follow : 

\- 'README.md' file describing the project 

\- '.here' dummy file

\- 'LICENSE.md file describing the chosen license

\- 'data' contains the raw data

\- 'outputs' contains results

\- 'analyses' contains R scripts

\- 'figures' contains figures

\- 'R' contains functions

\- 'DESCRIPTION' file

\- 'make.R' loads the package, dependencies and functions

\- '.gitignore' contains git to ignore when commiting
